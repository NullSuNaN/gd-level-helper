# Geometry Dash Level Processing Helper

Host this with `make`(or `python3 -m http.server 80` on windows).

For a temporary edit to the level data, you can use the `Parse Decoded As Object` button and operate it in the devtools, then use `Write To Decoded` button to write it back.

Online features probably won't be implemented.

This Project uses a parsing library [`GDParse`](https://github.com/HectLM/GDParse).

The library may not work properly.
